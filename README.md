# Detection-Engineering-Foundations-Lab

## Overview

This repository documents my Week 3 Detection Engineering learning journey. The project focuses on understanding how SOC analysts build, validate, and improve security detections using Windows event logs, Sysmon telemetry, Sigma rules, and the MITRE ATT&CK Framework.

Throughout the week, I explored the complete detection engineering lifecycle—from learning detection fundamentals and developing Sigma rules to analyzing false positives, mapping detections to MITRE ATT&CK, and assessing overall detection coverage.

---

# Objectives

- Understand Detection Engineering fundamentals.
- Develop Sigma detection rules for common attacker techniques.
- Investigate Windows endpoint activities using event logs and Sysmon.
- Differentiate true positives from false positives.
- Map detections to the MITRE ATT&CK Framework.
- Assess detection coverage and identify security gaps.
- Correlate multiple detections into attack timelines.
